### **1Ô∏è‚É£ Clone the Repository**  
```bash

```

### **2Ô∏è‚É£ Open the `index.html` Locally**

```bash

xdg-open index.html   # Linux

```

### **3Ô∏è‚É£ Paste & Beautify CK3 DNA**
- **Paste** your DNA string in the **input box**.
- Click **"Beautify DNA"** to process it.
